射影几何与范畴论中的对偶原理

发表于 2019-05-28 | 评论数：

最近接触到一点范畴论的内容, 范畴论里有个很有意思的东西叫"对偶原理"(duality principle), 熟悉射影几何的同学应该知道射影几何里也有一个对偶原理(其实线性代数里也有所谓对偶原理, 不过我第一时间想到的是射影几何里的那个), 查了一些资料之后, 发现这两个对偶原理有很强的联系, 所以稍微整理后写了这篇笔记.

射影几何中的对偶原理

射影平面上的对偶原理是广为人知的, 这里以丘维声《解析几何》中的描述为代表:

射影平面上的对偶原理 射影平面上, 如果一个图解命题\(\phi(Point, Line)\)是一条定理, 则它的对偶命题\(\phi(Line, Point)\)也可以证明是一条定理.

类似的, 对于一般的\(n\)维射影空间\(PG(n, \mathbb F)\), \(\mathbb F\)是一个域, \(r\)维子空间与\(n-r-1\)维子空间互为对偶, 可以得出相应的\(n\)维射影空间\(PG(n, \mathbb F)\)上的对偶原理.

范畴论中的对偶原理

范畴论中的对偶原理和反范畴的概念紧密相关, 反范畴的定义如下:

对于任意范畴\(\mathcal{C}\), 其反范畴\(\mathcal{C}^{op}\)定义如下:

  • 对象集: \(\mathrm{Ob} (\mathcal{C}^{op}) = \mathrm{Ob} (\mathcal{C})\);
  • 态射集: \(\forall X, Y,\mathrm{Hom}_{ \mathcal{C}^{op} }(X,Y) = \mathrm{Hom}_{ \mathcal{C} }(Y, X)\);
  • 态射\(f \in \mathrm{Hom}_{\mathcal{C}^{op}}(Y, Z), g \in \mathrm{Hom}_{\mathcal{C}^{op}}(X, Y)\)的合成\(f\circ^{op}g\) 定义为\(\mathcal C\) 中反向合成 \(g \circ f;\)
  • 恒同态射同\(\mathcal{C}\)

容易验证, \(\mathcal{C}^{op}\)是一个范畴, 且\((\mathcal{C}^{op})^{op} = \mathcal{C}\). 简而言之, 将一个范畴的箭头全部反转, 就得到了它的反范畴.

对于某个对象\(A\in \mathcal C\), 由于\(\mathrm{Ob}(\mathcal{C}^{op}) = \mathrm{Ob}(\mathcal{C})\), 因此\(\mathcal{C}^{op}\)中也有相同的对象\(A\), 记为\(A^{op}\). 同样, 对于态射集\(\mathrm{Hom}_{\mathcal{C}^{op}}(A^{op}, B^{op})\)记范畴\(\mathcal{C}\) 中态射\(f\) 对应的态射为\(f^{op}\). 在范畴\(\mathcal C\)和\(\mathcal{C}^{op}\)之间有对偶函子\(F\), 满足\(F(A) = A^{op},\ F(f) = f^{op},\ F(f \circ g) = g \circ f\).

由于反转箭头后范畴论诸公理仍然成立, 因此范畴\(\mathcal C\)中一条关于\(\mathrm{Ob}(\mathcal{C})\)和\(\mathrm{Mor}(\mathcal{C})\)的性质\(S\), 在对偶函子\(F\)作用下, 将对象\(A\), 态射\(f\) 替换为\(F(A),\ F(f)\)后的性质仍然成立, 记为\(S^{op}\), \(S^{op}\)是\(\mathcal{C}^{op}\)的一条性质. 这就是范畴论中的对偶原理.

两个对偶原理的联系

为了阐明射影几何的对偶原理背后的范畴论背景, 我们首先要建立起射影几何范畴的概念. 在此之前, 我们先给出一些之后会用到的关于\(n\) 维射影几何的概念, 如果读者预先已经对此有所了解, 可以跳过这些内容.

n维射影空间与n维射影几何

定义1 设\(V^{n+1}\)是域\(\mathbb F\)上的一个\(n+1\)维线性空间, \(V^{n+1}\)的全体非零向量构成的集合是一个\(n\)维射影空间, 记为\(P^n\). 设\(v \in V^{n+1},\ v \neq 0,\ k\in \mathbb F,\ k \neq 0\), \(kv\)与\(v\)表示\(P^n\)中的同一个点.

定义2 设\(M\)是\(V^{n+1}\)的一个非零子空间, 则\(M\)的非零向量的集合\(M_P\)是\(P^n\)的子空间.

定义3 \(P^n\)及其子空间的维数记为\(\mathrm{Pdim}\), \(\mathrm{Pdim} P^n = \dim V^{n+1}-1\), \(\mathrm{Pdim} M_P = \dim M - 1\). 规定\(\mathrm{Pdim}\{0\} = -1\). 0, 1, 2维的子空间分别称为点, 直线和平面, \(P^n\) 中\(n-1\)维子空间称为\(n-1\)维超平面.

定义4 定义\(P^n\)的子空间\(M_P, N_P\)的和与交为\(M_P + N_P = (M+N)_P,\ M_P \cap N_P = (M \cap N)_P\).

定义5 记\(\mathscr{P}(P^n)\)为\(P^n\)的全体子空间的集合, 称\(\mathscr{P}(P^n)\)是\(V^{n+1}\)上的一个\(n\)维射影几何.

射影几何范畴

定义6 射影几何范畴\(\mathbf{Prg}(P^n)\)构造如下:

  • 设\(P^n\)是一个\(n\)维射影空间, \(\mathbf{Prg}(P^n)\)以\(P^n\)的子空间为对象, \(\mathrm{Ob}(\mathbf{Prg}(P^n)) = \mathscr{P}(P^n)\).
  • 规定\(\mathrm{Hom}(M_P, N_P)=\{f_{MN}\}\)是一个单点集, 如果\(M_P \subseteq N_P\), \(\mathrm{Hom}(M_P, N_P)=\varnothing\), 如果\(M_P \nsubseteq N_P\). 全体态射组成\(\mathbf{Prg}(P^n)\)的态射集\(\mathrm{Mor}(\mathbf{Prg}(P^n))\).
  • 对于每个\(M_P \in \mathrm{Ob}(\mathbf{Prg}(P^n))\)规定\(\mathrm{Hom}(M_P, M_P)=\{\mathrm{id}_M\}\) 是由\(M_P\)上恒等态射组成的单点集.
  • 对于任意\(M_P\subseteq N_P \subseteq Q_P\), 定义合成映射 \(\circ\) 使得\(f_{NQ} \circ f_{MN} = f_{MQ}\), 对于不满足这一条件的子空间不存在态射\(f_{MQ}\), 故这一定义给出了所有可能的态射的合成.

下面我们利用范畴论的对偶原理证明射影几何的对偶原理, 这里的证明来自于这篇论文1, 这里修改了部分记法和符号, 并且改正了几个印刷错误, 某些细节有所调整.

命题1 设\(M_P \in \mathrm{Ob}(\mathbf{Prg}(P^n))\), 则\(\mathrm{Pdim}M_P=n-1-\mathrm{Pdim}M_P^{op}\).

证明:

设\(M_P\)是\(V^{n+1}\)的\(r\)维子空间, \(M_P = \mathrm{span}\{m_1, \cdots, m_r\},\ m_1, \cdots, m_r\)是\(M_P\)的一组基.

将\(\{m_1, \cdots, m_r\}\)扩充为\(V^{n+1}\)的一组基\(\{m_1, \cdots, m_r, m_{r+1}, \cdots, m_{n+1}\}\) , 并且令\(M_P^i = \mathrm{span}\{m_1, \cdots, m_i\},\ i=1,2,\cdots, n+1,\ M_P^0 = \{0\}\) .

\(M_P^i\)是\(P^n\)的一个\(i-1\)维子空间, 并且有链 \[ \{0\}_P = M_P^0 \subseteq M_P^1 \subseteq M_P^2 \subseteq \cdots \subseteq M_P^n \subseteq M_P^{n+1} = P^n. \] 如前所述, 反范畴是将范畴的所有箭头反转得到的, 因此对链中的每个对象施加对偶算子后我们有: \[ \{0\}_P^{op} \supseteq M_P^0 \supseteq (M_P^1)^{op} \supseteq (M_P^2)^{op} \supseteq \cdots \supseteq (M_P^n)^{op} \supseteq (M_P^{n+1})^{op} \supseteq (P^n)^{op}. \]

当\(i < j\)时, \(M_P^i \subsetneqq M_P^j\), 于是\((M_P^i)^{op} \supsetneqq (M_P^j)^{op}\). 又由于\(-1 \leq \mathrm{Pdim} (M_P^j)^{op} < \mathrm{Pdim} (M_P^i)^{op} \leq n\), 可得\(\mathrm{Pdim}\ (M_P^i)^{op} = n - i\).

又由于\(\mathrm{Pdim} M_P^i = i - 1\). 代入上式可得\(\mathrm{Pdim}M_P=n-1-\mathrm{Pdim}M_P^{op}\). \(\Box\)

命题2 在范畴\(\mathbf{Prg}(P^n)\) 中\(+,\ \cap\)两者互为对偶.

证明:

任取\(M_P, N_P \in \mathrm{Ob}(\mathbf{Prg}(P^n))\)由于\((\mathcal{C}^{op})^{op}=\mathcal{C}\), 命题等价于证明\((M_P \cap N_P)^{op} = M_P^{op} + N_P^{op}\)且\((M_P + N_P)^{op} = M_P^{op} \cap N_P^{op}\).

因为\(M_P \cap N_P \subseteq M_P\), 且\(M_P \cap N_P \subseteq N_P\), 于是\((M_P \cap N_P)^{op} \supseteq M_P^{op}\), 且\((M_P \cap N_P)^{op} \supseteq N_P^{op}\), 故而有\(M_P^{op} + N_P^{op} \subseteq (M_P \cap N_P)^{op}\).

因为\(M_P^{op} \subseteq M_P^{op} + N_P^{op}\)且\(N_P^{op} \subseteq M_P^{op} + N_P^{op}\), 于是\(M_P \supseteq (M_P^{op} + N_P^{op})^{op}\)且\(N_P \supseteq (M_P^{op} + N_P^{op})^{op}\), 故而有\((M_P^{op} + N_P^{op})^{op} \subseteq M_P \cap N_P\), \((M_P \cap N_P)^{op} \subseteq M_P^{op} + N_P^{op}\), 所以\((M_P \cap N_P)^{op} = M_P^{op} + N_P^{op}\).

于是\((M_P^{op} \cap N_P^{op})^{op} = (M_P^{op})^{op} + (N_P^{op})^{op} = M_P + N_P\), 所以\((M_P + N_P)^{op} = M_P^{op} \cap N_P^{op}\). 命题得证.\(\Box\)

对于某个\(n\)维射影空间里的图解命题, 通过将命题中的\(M \subseteq N,\ \cap,\ +\) 替换为对偶元素\(N^{op} \subseteq M^{op},\ +,\ \cap\)就可以得到对偶命题, 由命题2和范畴论中的对偶原理, 可以以得出对偶命题的正确性. 此外, 对于某个确定的射影几何范畴\(\mathbf{Prg}(P^n)\), 由命题1可知\(r\)维子空间\(M\)的对偶的维数为\(n-r-1\). 由这两点可知, 上述\(n\)维射影空间\(PG(n, \mathbb F)\)上的对偶原理成立.

附记

实际上, 利用线性空间的对偶原理也可以得出射影几何中的对偶原理, 或者说这几个名字相同的原理本质上是一回事. 之所以这里选择从范畴论的角度而不是线性代数的角度考虑问题, 是因为线性代数中的对偶原理也是反范畴的对偶原理的结果2, 因此我就索性从最底层出发了. 关于一般线性空间的对偶有机会还会再写一篇笔记.
